# 清見台中央公園クイズラリー(プロトタイプ)

木更津市・清見台中央公園を周遊しながら木更津に関するクイズに答えるWebアプリ。
授業「問題解決技法」における公園活性化提案のデモ。要件は [REQUIREMENTS.md](./REQUIREMENTS.md) を参照。

## 開発

```bash
npm install
npm run dev      # 開発サーバー (http://localhost:5173)
npm run build    # 本番ビルド (dist/)
npm run preview  # ビルド結果の確認
```

## 遊び方

- ホーム画面のスポットカードをタップするとクイズが始まる(デモ用)。
- 実運用では各スポットのQRコードが `…/?spot_id=hippo` のようなURLを指す。
  - スポットID: `hippo`(カバの置物) / `hall`(集会所) / `slide`(貯水タンク滑り台) / `clocktower`(時計台)
- 時計台以外を全部クリアすると時計台が解放され、最後にハンドルを回す演出が再生される。

## データの編集

クイズの問題・スポット定義は `src/data/spots.json` にまとまっている。
問題の追加・修正はこのファイルを編集するだけでよい。

- `park.lat` / `park.lng` は概算値。**正確な公園の座標に要修正**。
- `park.geoCheckEnabled: false` にすると位置情報チェック(ゆるい注意バナー)を無効化できる。
- 時計台の建設年は1974年(確認済み)。その他のクイズの事実関係は発表前に要確認。

## 進捗データ

スタンプ進捗はブラウザのlocalStorageのみに保存(サーバーなし)。
「はじめからやり直す」ボタン、またはブラウザのサイトデータ削除でリセットできる。
